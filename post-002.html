User Accounts Will Be Required for Filing Annual Reports

April 2, 2021

With the current Sunbiz.org system, users are not required to provide a username and password for business filings and some data downloads. The modernized Sunbiz site, however, will require users to create individual user accounts, each with a unique username and strong password for secure authentication. User accounts will also be required for filing an Annual Report. Also, users that want to file an Annual Report that makes changes to their business entity will be required to have filing authority.
